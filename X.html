FAQ :


❔    How can I tell if what I download is safe?

  • For files under 650MB, you can upload these to be scanned on VirusTotal and then use this guide to determine if any detections are most likely real or false positives.

  • Always check if the file you are downloading has the correct file-ending:
    • Video-Files:
      • .mp4 - MPEG-4 Part 14, widely used for digital video streams
      • .avi - Audio Video Interleave, introduced by Microsoft
      • .mov - Apple QuickTime Movie, used natively by the QuickTime framework
      • .wmv - Windows Media Video, developed by Microsoft
      • .mkv - Matroska Video, a flexible, open standard video file format
      • .flv - Flash Video, used by Adobe Flash Player
      • .webm - WebM, designed for use with HTML5 video
    • Audio-Files:
      • .mp3 - MPEG-1 Audio Layer III, widely used for compressing audio
      • .flac - Free Lossless Audio Codec, for lossless compression
      • .mp4 - Can also contain audio tracks in addition to video
      • .opus - A highly versatile audio codec, efficient for both music and speech
      • .wav - Waveform Audio File Format, a raw audio format created by Microsoft and IBM
      • .aac - Advanced Audio Coding, standardized, lossy compression
      • .m4a - MPEG-4 Audio, often found in Apple's iTunes Store
    • Text-Files:
      • .pdf - Portable Document Format, widely used for documents
      • .txt - Plain text file, contains no formatting
      • .doc - Microsoft Word Document, earlier version format
      • .odt - OpenDocument Text, used by LibreOffice/OpenOffice
      • .docx - Microsoft Word Open XML Document, newer version format
      • .rtf - Rich Text Format, supports rich text
      • .html - HyperText Markup Language, used for web pages

  • These are just the most common; the 'Use your Brain Doctrine' stays active.

  • For PC-Games there are usually two possibilities:
    • .Iso file
    • "preinstalled" folder

❔    Downloads, Torrents, & VPNs

  • Direct downloads are kept on the server from which you are retrieving the material. A secure browser like Firefox + uBlock Origin is all you need.

  • Torrents are peer-to-peer file sharing systems that need seeders (users who have previously downloaded the file). To begin torrenting files, a separate application is required.

  • Torrenting without proper protection may get you in trouble because your IP address is visible due to the nature of the protocol. Direct downloads generally don't require additional protection.
  • VPN is an abbreviation for virtual private network, which is a service that secures your internet connection and online privacy. It encrypts your data by concealing your IP address, allowing you to circumvent worldwide limitations on website access while still safeguarding your true identity.

  • When you connect to a VPN, your internet traffic is routed through a VPN server before being sent to your destination.

  • Free VPNs are more likely to collect data from you for marketing purposes, be too sluggish to use, or not enable torrent activity at all. It is advised to subscribe to a VPN service such as Mullvad or ProtonVPN.

❔    Common Mistakes

  • "I can't open that "rar" file, what is it?"
    A ".rar" file is a compressed archive file created using the RAR compression format. These files contain one or more files and are often used to reduce file sizes for storage and sharing. To open a ".rar" file, you typically need special software, such as Winrar., that can extract the files from the archive.

  • "What do I do with this .Iso file?"
    An ISO file is a complete copy of the contents of a physical disc, used for installing software, creating backups, and more. To use it, you can mount or extract it, create bootable media, or use it in virtual machines. If you use Windows 10 or later no additional Software is needed to open a .Iso.

  • "I installed that game correctly, but every time I start it it just opens Steam/Epic?"
    That means you did not copy your cracked files to the right place, or did not copy them at all.
    Check the readme, the files you downloaded or the page you downloaded from to find the cracked files.
